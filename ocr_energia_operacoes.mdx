## Ordem de implantação

### Banco de dados

Executar nesta sequência:

1. `V360.0002.CONTA_ENERGIA_DOCUMENTO.sql`
2. `V360.0003.CONTA_ENERGIA_CONSUMO.sql`
3. `V360.0004.ALTER_CONTA_ENERGIA_CONSUMO_OCR.sql`
4. `V360.0007.ALTER_CONTA_ENERGIA_DOCUMENTO_ETAPA.sql`

### n8n

Importar nesta ordem:

1. `OCREnergia _ Download Arquivo.json`
2. `OCREnergia _ Recognition.json`
3. `OCREnergia _ Envio Email.json`
4. `OCREnergia _ Orquestrador.json`

## Troubleshooting

### Documento não gerou consumo

Verificar:

- se o documento foi bloqueado antes por ausência de `IDMOV`
- se o OCR retornou `PROCESSED`
- se o parser gerou itens válidos
- se os itens não foram descartados pelo filtro de total
- se houve vínculo com `GFILIAL` (ATIVO = 1) e `TMOVCOMPL`

## Reprocessamento controlado

### Quando o IDV360 foi corrigido no RM

Exemplo:

```sql
UPDATE dbo.TMOVCOMPL
   SET IDV360 = 12345
 WHERE CODCOLIGADA = 1
   AND IDMOV = 98765;
```

### Playbook A: Documento bloqueado por falta de IDMOV

Esse cenário ocorre quando o orquestrador valida `TMOVCOMPL.IDV360`, não encontra `IDMOV` e o fluxo não baixa PDF nem chama `Recognition`.

```sql
UPDATE dbo.CONTA_ENERGIA_DOCUMENTO
   SET TRATADA = 0,
       STATUS_DOCUMENTO = 'PENDENTE',
       STATUS_RECOGNITION = NULL,
       ETAPA = 'ENCONTRADO',
       ETAPA_ERRO = NULL,
       MENSAGEM_ERRO = NULL,
       MOTIVO_ERRO = NULL,
       RECOGNITION_DOCUMENT_ID = NULL,
       RECOGNITION_PAYLOAD = NULL,
       RECMODIFIEDBY = 'n8n.integracao',
       RECMODIFIEDON = GETDATE()
 WHERE ID_V360_DOCUMENTO = 12345;
```

### Playbook B: Documento já possui RECOGNITION\_DOCUMENT\_ID

Esse cenário ocorre quando o PDF já foi enviado, o `RECOGNITION_DOCUMENT_ID` existe e você quer reprocessar sem reenviar o arquivo.

```sql
DECLARE @ID_DOC BIGINT;

SELECT @ID_DOC = ID
FROM dbo.CONTA_ENERGIA_DOCUMENTO
WHERE ID_V360_DOCUMENTO = 12345;

DELETE FROM dbo.CONTA_ENERGIA_CONSUMO
WHERE ID_DOC = @ID_DOC;

UPDATE dbo.CONTA_ENERGIA_DOCUMENTO
   SET TRATADA = 0,
       STATUS_DOCUMENTO = 'PENDENTE',
       ETAPA = 'ENCONTRADO',
       ETAPA_ERRO = NULL,
       MENSAGEM_ERRO = NULL,
       MOTIVO_ERRO = NULL,
       RECMODIFIEDBY = 'n8n.integracao',
       RECMODIFIEDON = GETDATE()
 WHERE ID_V360_DOCUMENTO = 12345;
```
